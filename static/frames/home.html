  • List Records

    GET /do/check_coll?coll=[coll_id]

  • PHP Example 

    $client= new HttpClient('cucudb.duapp.com');
$coll_id = 6;
$client->get('/do/check_coll?coll='. $coll_id);
$coll = json_decode($client->getContent(), true);
print_r($coll['records']);	// array of records

    NodeJS Example 

    var request = require('request');
var coll_id = 6;
request({
	method: 'GET',
	url: 'http://cucudb.duapp.com/do/check_coll?coll='+ coll_id
}, function(err, res, body){
	var coll = JSON.parse(body);
	console.log(coll['records']);	// array of records
});

  • Add Record

    POST /do/add_record?coll=[coll_id]

    Parameters
    • field1: value1
    • field2: value2
    • field3: value3
    • ...

  • PHP Example 

    $client= new HttpClient('cucudb.duapp.com');
$coll_id = 6;
$client->post('/do/add_record?coll='. $coll_id, array(
	'message' => 'are you crazy?',
	'from' => 'Peter',
	'to' => 'Mary'
));
$reply = json_decode($client->getContent(), true);
print_r($reply);	// reply object

    NodeJS Example 

    var request = require('request');
var coll_id = 6;
request.post({
	method: 'POST',
	url: 'http://cucudb.duapp.com/do/add_record?coll='+ coll_id,
	form: {
		'message': 'are you crazy?',
		'from': 'Peter',
		'to': 'Mary'
	}
}, function(err, res, body){
	var reply = JSON.parse(body);
	console.log(reply);	// reply object
});
